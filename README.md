Легковесный инструмент для управления диаграммами через консоль. Создан для быстрой интеграции в любые React-проекты.

Управление через консоль - все команды вводятся в терминале
3 типа диаграмм - столбчатая, круговая, линейная
Автообновление - диаграмма обновляется мгновенно после команд
Портативность - мелкий и легко вставить для использования в своих проектах
Минимализм - никакого лишнего дизайна, только диаграмма

Запуск через 2 терминала:
npm run dev
npm run console


Команды:
do 1:	     Столбчатая диаграмма
do 2:        Круговая диаграмма
do 3:	     Линейная диаграмма
add 100-Имя: Добавить/обновить запись, add 500-Продажи
del Имя:     Удалить запись, del Продажи
list:        Показать все записи
clear:       Очистить все данные
help:      	 Показать справку
exit:      	 Выйти

Для изменения цветов В main.jsx найдите массив colors и измените по rgba.
Для изменения размера В main.jsx измените стили контейнера.

Если инструмент полезен, поставьте звезду <3

A lightweight tool for managing charts through the console. Created for quick integration into any React projects.

 Console-based management - all commands are entered in the terminal
3 types of charts - bar, pie, and line charts
Auto-refresh - the chart updates instantly after commands
Portability - small and easy to insert into your projects
Minimalism - no unnecessary design, just a chart

Running through 2 terminals:
npm run dev
npm run console

Commands:
do 1: Bar chart
do 2: Pie chart
do 3: Line chart
add 100-Name: Add/update record, add 500-Sales
del Name: Delete record, del Sales
list: Show all records
clear: Clear all data
help: Show help
exit: Exit

To change colors In main.jsx, find the colors array and change it to rgba.
To change the size In main.jsx, change the container styles.
